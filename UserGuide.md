### 8.1 小程序前端

| 目录/文件    | 描述                                             |
| ------------ | ------------------------------------------------ |
| components/  | 自定义组件目录                                   |
| icon/        | 图标文件目录                                     |
| lib/         | 使用的第三方UI库（iview、vant）目录              |
| pages/       | 页面源代码目录，每个页面包括wxml、js、wxss、json |
| utils/       | 工具脚本目录，包括登录、交互和一些常用工具       |
| app.js       | 全局数据和函数                                   |
| app.json     | 全局页面、组件配置                               |
| app.wxss     | 全局样式                                         |
| README.md    | 自述文件，包含页面详细说明                       |
| sitemap.json | 小程序索引配置                                   |
| theme.json   | 小程序全局主题样式                               |
