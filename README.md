## 每周安排

* 周五：课上汇报
* 周六至周日下午：制定初步分工计划，着手开发（视任务量而定）
* 周日晚：晚上会议讨论核心问题，细化及调整分工
* 周一至周四下午：完成开发任务
* 周四晚：提交周报及相应文档、完成汇报PPT
